# Contributing

Guidelines for group contributions:

- Fork / clone the repository and create feature branches.
- Follow the directory structure: `content/modules/moduleX` for module materials, `src/components` for shared UI.
- Keep commits small and focused; write clear commit messages.
- Add or update tests under `tests/` when adding functionality.
- Open a PR and request review from another team member.

Optional: follow a simple feature naming convention: `feature/module5-quizzes`.
